# Índice Geral do Projeto

Este arquivo centraliza a navegação dos materiais criados no projeto.

## 1) Scripts principais
- `parte_a_eda.py` -> análise exploratória inicial (EDA) para a Parte A.
- `draw_yolo_boxes_v1.py` -> v1: desenha caixas YOLO em imagens.
- `draw_yolo_boxes_v2.py` -> v2: desenha caixas + CSV consolidado.
- `draw_yolo_boxes_v3.py` -> v3: organização por execução + CSV por imagem opcional.

## 2) Guias de execução
- `COMO_RODAR_DRAW_YOLO_BOXES.md` -> comandos e parâmetros para v1, v2, v3.

## 3) Diagramas de fluxo (Mermaid)
- `DIAGRAMAS_MERMAID_DRAW_YOLO_BOXES.md` -> fluxos v1, v2 e v3.

## 4) Documentação detalhada (organizada)
Pasta: `docs_draw_yolo/`

- `docs_draw_yolo/README.md`
- `docs_draw_yolo/EXPLICACAO_DETALHADA_DRAW_YOLO_BOXES.md`
- `docs_draw_yolo/EXPLICACAO_DETALHADA_DRAW_YOLO_BOXES_V1.md`
- `docs_draw_yolo/EXPLICACAO_DETALHADA_DRAW_YOLO_BOXES_V2.md`
- `docs_draw_yolo/EXPLICACAO_DETALHADA_DRAW_YOLO_BOXES_V3.md`

## 5) Entrega da Parte A (disciplina)
Pasta: `entrega_parte_a/`

- `entrega_parte_a/roteiro_slides_parte_a.md`
- `entrega_parte_a/checklist_validacao_parte_a.md`
- `entrega_parte_a/plano_comparativo_futuro.md`

## 6) Saídas geradas (outputs)
- `outputs/parte_a_eda/` -> figuras e sumário da EDA.
- `outputs/label_box_preview_v1/` -> resultados da v1.
- `outputs/label_box_preview_v2/` e `outputs/label_box_preview_v2_train/` -> resultados da v2.
- `outputs/yolo_boxes_v3_runs/` -> execuções organizadas da v3.

## Sugestão de uso
1. Começar por `INDEX_GERAL.md`.
2. Executar com base em `COMO_RODAR_DRAW_YOLO_BOXES.md`.
3. Consultar detalhes técnicos em `docs_draw_yolo/`.
4. Usar Mermaid para explicar fluxo em apresentação.
